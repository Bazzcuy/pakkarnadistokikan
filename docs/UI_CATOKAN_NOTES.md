# UI CATOKAN Notes

Fokus penyegaran UI:
1. Branding yang kuat (logo + banner + nama CATOKAN).
2. Warna biru–tosca dengan aksen oranye agar terasa fresh dan tetap kontras.
3. Dashboard yang cepat dipahami saat presentasi: ada stat card dan menu utama.
4. Form dibuat lebih ramah UX: label jelas, tombol besar, padding lega.
5. Layout tetap responsif di HP karena semua konten dibungkus ScrollView vertikal.
