# UI Premium CATOKAN - Figma Style

Perubahan pada versi ini hanya difokuskan pada tampilan dan pengalaman pengguna Android agar lebih mendekati rancangan Figma yang sudah disetujui.

## Perubahan utama
1. Mengganti aset logo dan banner CATOKAN dengan gaya premium ocean theme.
2. Memperbarui warna utama menjadi biru-teal dengan aksen coral.
3. Memperbarui dashboard agar lebih modern: hero banner, shortcut aktivitas, stat card, menu utama, dan bottom navigation.
4. Memperbarui halaman login, splash, onboarding, input stok, produksi, dan penjualan agar lebih bersih dan nyaman dipakai.
5. Mempertahankan ID komponen, class Java, database SQLite, dan logika bisnis agar fitur tidak rusak.

## Catatan penting
- Revisi ini tidak mengubah skema database utama.
- Revisi ini tidak mengubah service stok, produksi, penjualan, laporan, backup, maupun validasi.
- Jika ingin membuat APK baru, buka folder `android-native` di Android Studio lalu build ulang APK.
